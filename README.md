## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/samson-16/Open_role.git
   cd Open_role
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```
